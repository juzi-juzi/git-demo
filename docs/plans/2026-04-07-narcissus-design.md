# 三位水仙花判断程序 Design

**目标**

写一个独立的 Python 脚本，从标准输入读取一个整数，并判断它是否为三位水仙花数。

**范围**

- 只判断三位数。
- 非三位数统一输出“不是水仙花数”。
- 使用数字运算拆分百位、十位、个位。

**方案对比**

1. 数字运算拆位
   直接用整除和取模得到三位数字，计算立方和后与原数比较。
   这是推荐方案，逻辑直接，和题意一致。

2. 字符串拆位
   把输入转成字符串后逐位转整数求和。
   实现也可行，但相对不如数字运算贴合题目。

**最终设计**

程序读取一个整数 `n`。

- 如果 `n` 不在 `100` 到 `999` 之间，输出 `不是水仙花数`
- 否则计算：
  - 百位：`n // 100`
  - 十位：`n // 10 % 10`
  - 个位：`n % 10`
- 若三位数字的立方和等于 `n`，输出 `是水仙花数`
- 否则输出 `不是水仙花数`

**错误处理**

为了保持题目级别的简洁实现，默认输入是合法整数，不额外处理非数字输入。

**测试**

至少覆盖：

- 输入 `153` 时输出 `是水仙花数`
- 输入 `123` 时输出 `不是水仙花数`
- 输入非三位数如 `99` 时输出 `不是水仙花数`
